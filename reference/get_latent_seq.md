# get index of latent status

get index of latent status

## Usage

``` r
get_latent_seq(cause_list, ord, select_latent = NULL, exact = TRUE)
```

## Arguments

- cause_list:

  see mode_options in
  [`nplcm()`](https://zhenkewu.com/baker/reference/nplcm.md)

- ord:

  order of cause_list according to posterior mean

- select_latent:

  Default is NULL

- exact:

  Default is TRUE

## Value

a vector of indices
