# Get position to store in data_nplcm\$Mobs:

Get position to store in data_nplcm\$Mobs:

## Usage

``` r
lookup_quality(quality_nm)
```

## Arguments

- quality_nm:

  names of quality: can be "BrS", "SS" or "GS"

## Value

position of the quality name: "BrS"-1; "SS"-2; "GS"-3.

## Details

also works for a vector
